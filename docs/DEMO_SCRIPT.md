# 기말 과제 데모 촬영 시나리오

## 반드시 보여 줄 장면

1. GitHub 저장소와 성공한 Actions 실행
2. AWS EC2 터미널의 `docker compose ps`
3. 두 컨테이너가 `healthy`이고 Streamlit·FastAPI가 분리 실행 중인 상태
4. FastAPI `/health` 응답
5. 외부 EC2 주소로 FlowPilot Streamlit 접속
6. 사용자 입력 7개를 변경
7. **내 몰입 루틴 생성하기 →** 버튼 실행
8. 추천 카드, 선택 근거, 점수, 타임라인, 대안, 팁 표시
9. EC2 터미널에서 추천 API JSON의 `recommendation`, `timeline`, `alternatives` 확인
10. FastAPI 로그에서 Streamlit 컨테이너가 보낸 `POST /api/v1/recommend 200` 확인

## 촬영용 입력 예시

- 목표: `기말 프로젝트 README와 배포 검증을 마무리하기`
- 가용 시간: `75분`
- 에너지: `4/5`
- 환경: `약간 소란스러움`
- 작업 유형: `구현·문제 해결`
- 방해 수준: `보통`
- 선호 리듬: `집중과 휴식의 균형`
